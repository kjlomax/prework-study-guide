HTML

  • Enter your HTML notes here
  • The head element contains information about the webpage not visible to the user.
  • The body element represents the visible content shown to the user.

CSS

  • CSS code goes inside the head element.
  • Classes are css selectors to style specific or multiple html elements with the same css styles.
  • There are three ways to connect css to an html file:
    1. Internal: add style element inside of head element
    2. External: inside head tag, add link element with rel="stylesheet" and href="(the name of the css file)"
    3. Inline: add style attributes to the html elements directly.

Git

  • All commands begin with git.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone repositoryUrl: clones a remote repository to current location.

JavaScript

  • Javascript makes a website interactive.
  • The interactivity lives between the script tags.
